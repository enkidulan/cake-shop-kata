Pre-requirements
-----------------

CMD interaction with the project is done via `pipenv <https://python-poetry.org/>`_, make
sure you have it installed:

.. code-block:: bash

    pip install --user poetry

Setup
---------------------

To install dependencies run:

.. code-block:: bash

    poetry install



Shell
---------------------

.. code-block:: bash

    poetry shell

Tests
---------------------

.. code-block:: bash

    pytest tests/
